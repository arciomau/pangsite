Скрыпт на Python

Скрыпт для праверкі беларускіх панграм. Напісаны на мове Python, працуе ў версіі 2.6 і навейшых (таксама і ў Python 3). Мае просты графічны інтэрфейс (tkinter).

Сістэмныя патрабаванні: любая АС, Python 2.6 ці навейшыя версіі, модуль tkinter (па змоўчанні ў Windows, а ў Linux — альбо асобным пакетам, альбо з адпаведным USE-сцягам, у залежнасці ад дыстрыбутыву).
